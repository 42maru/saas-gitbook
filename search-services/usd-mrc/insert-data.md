# Upload passage

## Browser

문서 추가는 Documents 화면에서 가능합니다.

![](../../.gitbook/assets/image.png)

그 다음 File Upload 버튼을 클릭 하시면 새로운 창이 팝업되며 이곳에서 드래그 앤 드롭으로 파일을 업로드 하실 수 있습니다.

{% hint style="info" %}
json, excel 형식의 파일을 지원합니다.
{% endhint %}

![](../../.gitbook/assets/image-11.png)
